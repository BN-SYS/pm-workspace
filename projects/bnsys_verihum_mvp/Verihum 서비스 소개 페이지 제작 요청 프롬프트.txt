# Verihum 서비스 소개 페이지 제작 요청

## 서비스 개요

- 서비스명: Verihum (Verify + Human)
- 분류: AI 기반 사람 작성 인증 서비스 (SaaS)
- 핵심 개념: 결과물이 아닌 작성 과정을 기반으로, AI가 작성했는지 사람이 직접 작성했는지를 판별하고 인증한다

## 배경 및 문제

- 생성형 AI 확산으로 교육 현장에서 학생이 과제를 AI로 제출하는 사례 급증
- 기존 AI 탐지 방식은 결과물 비교 기반으로 정확성·확장성에 한계
- 교사들이 수행평가 채점에 과부하를 겪고 있으며, 생기부 반영 특성상 수행평가의 공정성이 중요
- 해결 방향: 과정 기반 평가로 작성 주체를 판별하고, 선생님의 업무 부담을 경감

## 솔루션

- 작성 과정을 자동 기록하고 AI로 분석해 사람 작성 여부를 인증
- 과정 재현 기능 제공 (평가자가 작성 과정을 직접 확인 가능)
- 인증 리포트 자동 발급

## 목표 시장 및 수익 모델

- 1차: 초·중·고 교육기관 (수행평가·생기부)
- 2차: 대학, 온라인 교육 플랫폼, 자격시험·공인 시험기관
- 3차: 기업 채용 과제, 공공기관
- 수익 모델: SaaS 구독형 / 사용량 기반 과금 / API 연동 / B2B·B2G 인증 서비스

## 제작 조건

### 필수 반영
- 결과물이 아닌 과정 기반 인증임을 명확히 전달
- 선생님의 수행평가 업무 부담 경감 메시지 포함
- CTA: 도입 문의하기 (비엔시스템 문의 링크)
- 반응형 (모바일 대응)
- 단일 HTML 파일로 제작

### 절대 포함 금지 (특허 보호)
- 기술 구현 방식 세부사항 일체 노출 금지
  - 예: 타이핑 속도, 타이핑 리듬, 키 입력 패턴, 수정 흐름, 붙여넣기 감지 방식 등
- 제너럴한 표현만 사용할 것 ("작성 과정 기록", "AI 분석", "인증" 수준으로 표현)
