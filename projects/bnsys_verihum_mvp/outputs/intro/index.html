Solution

작성 과정 데이터로

Verihum은 글을 쓰는 과정 자체를 분석한다. 작성 행동 데이터를 기반으로 AI 작성 여부를 판별하고 공식 인증한다.

교사는 결과물이 아닌 과정 리포트로 채점한다. 업무 부담은 줄이고, 평가의 신뢰성은 높인다.

작성 과정 자동 기록

과제 작성 전 과정을 자동으로 수집한다. 별도 프로그램 설치 없이 웹 브라우저만으로 동작한다.

과정 리플레이

교사는 학생의 작성 과정을 그대로 재현해 확인할 수 있다. 결과물 너머, 작성의 맥락이 보인다.

AI 자동 분석

수집된 행동 데이터를 AI가 분석해 직접 작성 여부를 판별하고 과정 점수를 자동 산출한다. 교사의 채점 부담을 대폭 줄인다.

인증 리포트 발급

분석 결과를 기반으로 공식 인증 리포트를 발급한다. 수행평가·과제·시험 등 다양한 평가 환경에 바로 적용 가능하다.
