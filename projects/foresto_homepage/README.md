## 2. 회원 권한 구조

| 등급 | 설명 | 전환 방법 | 접근 범위 |
|------|------|-----------|-----------|
| 비회원 | 미로그인 상태 | — | 공개 정보 페이지, 공지사항, 교육 안내 |
| 일반회원 | 본인인증 후 가입 | 직접 가입 | 비회원 + 자유게시판, 일자리정보, 강좌 신청 |
| 정회원 | 교육수료 + 월 회비 납부 확인 | **관리자 수동 등급 변경** | 회원 + 역량강화·사공단·동아리 게시판 등 |
| 관리자 | 시스템 전체 관리 | 시스템 설정 | 전체 |

> **정회원 자동화 없음**: 교육 수료 및 회비 납부 확인 후 관리자가 수동으로 등급 변경  
> **본인인증 필수**: 회원가입 시 휴대폰 본인인증 (NICE 또는 이니시스, 건당 비용 발생)  
> → 업체 추천은 개발사가 제안하며, **계약 및 비용 처리는 고객사 직접 진행**

## 4. 핵심 기능 상세 명세

### 4-1. 협회 일정 캘린더

**개요**  
협회의 모든 일정을 한 화면에서 확인할 수 있는 캘린더 페이지.  
구글 캘린더 형태를 참고하며, 오픈소스 라이브러리 활용 우선 검토.

**기능 요구사항**


| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 월별 캘린더 뷰 | 이전/다음 월 이동, 일정 노출 | 필수 |
| 목록형 뷰 전환 | 월별 일정 리스트 형태 | 필수 |
| 일정 클릭 이동 | 제목 클릭 시 관련 게시물/URL로 이동 | 필수 |
| 단순 안내 일정 | 링크 없이 텍스트만 표시 | 필수 |
| 관리자 일정 등록 | 제목 + 날짜 + 링크(선택) 입력 | 필수 |
| 관리자 목록형 관리 | 관리자는 목록 형태로 입력/수정/삭제 | 필수 |
| 권한별 노출 | 일정별 권한 처리 가능 | 필수 |

**일정 카테고리**: 교육/강좌, 사회공헌 활동, 협회 회의, 동아리 모임, 행사

**접근 권한**: 전체 공개 (열람), 관리자만 등록·수정·삭제

### 4-2. 강좌 통합 신청 시스템 ⭐

**개요**  
전문가과정, 시민아카데미, 역량강화 특강, 숲해설 강사 신청 등 모든 강좌를 하나의 신청 흐름으로 통합 관리.

**강좌 개설 (관리자)**

| 필드 | 설명 | 비고 |
|------|------|------|
| 강좌명 | 텍스트 | 필수 |
| 교육 기간 | 시작일~종료일 | 필수 / 수료증에 자동 반영 |
| 기수 | 숫자 입력 | 전문가과정 해당 / 기수 기준 협의 필요 |
| 정원 | 숫자 | 선착순 마감 기준 |
| 카테고리 | 전문가과정/시민아카데미/역량강화 등 | — |
| 신청 가능 여부 | 오픈/마감 상태 | — |

**신청 흐름 (사용자)**

```
신청 안내 페이지
  → 개인정보 동의 (체크박스)
  → 신청 폼 (이름 / 연락처 / 이메일 / 지역 / 파일 업로드 / 기타)
  → 신청 완료 페이지
```

**신청자 관리 (관리자)**

| 기능 | 설명 |
|------|------|
| 신청자 목록 | 강좌별 신청자 조회 |
| 상태 변경 | 신청완료 / 수료 / 취소 |
| 엑셀 다운로드 | 신청자 목록 전체 추출 |
| 수료증 PDF | 관리자도 개별 다운로드 가능 |

**적용 강좌 목록**

- 숲해설가 전문가과정 (REQ-031)
- 시민아카데미 교육강좌 (REQ-033)
- 회원활동 역량강화 교육강좌 (REQ-035)
- 숲해설 강사 신청 (REQ-038 / 결과 공지 없음, 신청만 수집)
- 후원하기 신청폼 (REQ-057)

### 4-3. 수료증 PDF 발급 ⭐

**개요**  
관리자가 신청자 상태를 '수료'로 변경하면, 회원의 마이페이지에서 PDF 다운로드 버튼이 활성화되는 방식.

**수료증 포함 항목**

| 항목 | 데이터 출처 |
|------|-------------|
| 성명 | 회원 정보 |
| 과정명 | 강좌 데이터 |
| 교육 기간 | 강좌 데이터 (시작일~종료일) |
| 기수 | 강좌 데이터 (관리자 입력) |
| 발급일 | 시스템 날짜 |
| 발급 기관 | 고정값 (한국숲해설가협회) |

**처리 흐름**

```
관리자: 신청자 목록 → 상태 '수료'로 변경
  ↓
사용자 마이페이지 → 교육이수내역 → 수료 항목 옆 '수료증 다운로드' 버튼 활성화
  ↓
클릭 시 지정 포맷으로 PDF 생성 → 다운로드
```

**참고**: 기존 수료증은 HWP 파일 형태. 샘플 파일 수령 후 디자인 확정 필요.  
→ **고객사 액션**: HWP 샘플 및 전문가과정 수료증 실물 이미지 전달

## 5. 게시판 정책

### 5-1. 게시판 유형 분류

| 유형 | 설명 | 해당 게시판 |
|------|------|------------|
| 정보 페이지 (정보P) | 관리자가 퍼블리싱으로 작성, CMS 관리 없음 | 인사말, 미션, 연혁, 정관 등 |
| 일반 게시판 (게시판P) | 에디터 사용, 글/사진 등록 가능 | 공지사항, 동아리소식, 사공단소식 등 |
| 기능 페이지 (기능P) | 신청폼, 캘린더 등 개발 필요 | 교육신청, 협회일정, 수료증 발급 등 |

### 5-2. 댓글 정책

> **미확정 항목** — 고객사 확인 후 문서 업데이트 필요

- 오픈소스 사용 시 댓글 기능 기본 포함 가능
- 고객사 요청 방향: 댓글 기능 최소화 또는 특정 게시판에만 허용
- **확인 요청 사항**: 댓글 허용 게시판 목록 확정

### 5-3. 공지사항

- 작성 권한: 관리자 전용
- 열람 권한: 전체 공개 (비회원 포함)
- 고정 공지 기능 포함

### 5-4. 파일 업로드 정책

- 사공단 일지 등 파일 업로드 게시판: **용량 제한 설정 필수**
- 용량 초과 시 안내 문구 노출 (UX 개선)
- 허용 확장자: JPG, PNG, PDF, HWP, DOCX 등 (개발 시 확정)
- 권장 업로드: JPG 형식으로 사진 업로드 안내 문구 추가

## 7. 회원 데이터 마이그레이션 (개발 X)

**배경**: 기존 홈페이지의 회원 DB를 신규 시스템으로 이전 필요.
어르신 회원 비중이 높아 재가입 요청 시 이탈 우려.

**검토 방안**

| 방안 | 설명 | 비용 |
|------|------|------|
| 자동 마이그레이션 | DB 테이블 분석 후 직접 이전 | 높음 (추가 견적) |
| 엑셀 업로드 | 고객사가 기존 데이터를 신규 양식에 맞게 정리 후 일괄 업로드 | 낮음 |

**최종 확정: 마이그레이션 없이 전체 신규 회원가입**

기존 회원 일괄 등록 방식은 비밀번호 처리 등 추가 개발이 필요하여 범위 외로 제외.
기존 회원 포함 전원 신규 회원가입하는 방향으로 확정.

## 8. 구 홈페이지 유지

- 기존 홈페이지는 데이터 보존 목적으로 **유지**
- 신규 홈페이지 하단 푸터에 구 홈페이지 링크 제공
- 서브도메인 설정 가능 여부 확인 필요 (예: \`old.도메인.or.kr\`)
- 서버 유지 비용 별도 발생 → 금액 확인 후 고객사 결정
