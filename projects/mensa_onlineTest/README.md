## 요구사항 요약

기존 멘사코리아 입회테스트 사이트에 회원/비회원 모두 무료로 응시 가능한 3단계 종합 지능 테스트 기능을 추가 개발한다.<br>
(단, 대외적으로는 지능 또는 IQ테스트 임을 언급하지는 않는다)<br>
결과 확인 시 2단계 결제 옵션(기본 결과/상세 리포트)을 제공하며, 단계별 문제 은행 기반 랜덤 출제, 보기 순서 랜덤 셔플, 응시자 출생연도 정보를 기반으로 연령 보정 채점, 비정상 응시 패턴 차단, 관리자 문제/로그 관리 기능을 포함한다.<br>
관리자페이지는 기존에 만들어져 있는 사이트 또는 별도 링크에 '온라인테스트 관리' 메뉴를 추가 개발한다.

- 기존 입회 테스트 외 별도 온라인 체험형 콘텐츠 신설 (Mensa Korea Brain Challenge)

## 3. 테스트 구조 상세

### 3-1. 단계별 특징

| 단계 | 문제 수 | 시간 제한 | 측정 영역 | 문제 은행 규모 | 기본점수 | 문제당 배점 |
|------|---------|----------|----------|--------------|---------|-----------|
| 1단계 | 관리자 설정 | 관리자 설정 | 도형 추리 | 관리자 설정 N개 | 관리자 설정 | (100 - 기본점수) / 문제수 |
| 2단계 | 관리자 설정 | 관리자 설정 | 수리, 상식, 논리 | 관리자 설정 N개 | 관리자 설정 | (100 - 기본점수) / 문제수 |
| 3단계 | 관리자 설정 | 관리자 설정 | 기억력, 인지, 작업능력 | 관리자 설정 N개 | 관리자 설정 | (100 - 기본점수) / 문제수 |

#### 설정 예시 (관리자에서 조정 가능)

- 1단계: 15문제, 기본점수 40점, 문제당 4점, 제한시간 30분
- 2단계: 5문제, 기본점수 60점, 문제당 8점, 제한시간 10분
- 3단계: 5문제, 기본점수 60점, 문제당 8점, 제한시간 10분

### 3-2. 문제 출제 로직

**문제 은행 구조**

- 각 단계별로 독립적인 문제 은행(pool) 보유
- 예: 1단계 문제 은행에서 응시 시 랜덤 N개 선택
- 사용자마다 다른 문제 조합 출제
- 🔴 비정상 패턴 감지로 인한 단계 재시작 시에는 새로운 랜덤 문제 세트로 출제 (동일 문제 재출제 방지 권장)

**문제 등록 형식**

- 문제(질문)는 아래 3가지 형식 중 **1가지만** 선택하여 등록
  - 텍스트만
  - 파일첨부(이미지)만
  - 텍스트 + 파일첨부(이미지) 혼합
- 보기(선택지)는 **파일첨부(이미지)만** 가능 (최소 2개 ~ 최대 8개)

**채점 기준**

- 정답 채점은 **문항 ID**와 **보기 ID** 기반으로 수행
- 응시자가 선택한 보기 ID와 정답 보기 ID를 비교하여 정오 판별

**랜덤 처리 규칙**

- **문제 선택**: 문제 은행에서 랜덤 추출 후, 추출된 순서대로 출제 (출제 후 순서 변경 없음)
- **보기 개수**: 관리자가 문제 등록 시 입력한 보기의 수는 문제마다 최소 2개에서 최대 8개로 다를 수 있으며, 입력한 보기는 모두 제시
- **보기 순서**: 매 응시마다 랜덤 셔플 (보기 ID는 고정, 표시 순서만 변경)

### 3-3. 검증 로직 (비정상 응시 차단)

**각 단계 완료 후 검증 (관리자 설정값 기준)**

- 정답률 N% 미만 → 해당 단계 재응시
- 평균 응답시간 N초 미만 → 해당 단계 재응시
- 미통과 시 메시지: "정답률이 너무 낮거나 비정상적인 응시 패턴이 감지되었습니다. 문제를 충분히 읽고 신중히 응답해주세요."

**재응시 처리 원칙**

- 재응시 시 해당 단계의 문제 세트는 새롭게 랜덤 구성
- 🔴 비정상 응시 감지로 재시작되어도 **하나의 응시 로그**로 저장 (응시자 정보 유지, 재응시 횟수 및 비정상 플래그 누적 기록)

## 5. 점수 산출 로직

### 5-1. 점수 체계 개요

**100점 만점 체계**

- 각 단계별로 독립적인 점수 산출 (최대 100점)
- 종합 점수는 1단계(×3), 2단계(×1), 3단계(×1), 작업속도(×1), 메타인지(×1)의 **가중 평균 (/7)**
- 연령 보정은 단계별 점수 및 작업속도에 적용 (각 최대 100점 초과 불가)
- 종합 점수 산출 후 **오프라인 입회테스트 유도를 위한 전체 점수 상향 보정** 적용 (최대 100점 초과 불가)

**등급 체계**

- 결과 화면에 **등급(하위권 / 중위권 / 상위권 / 최상위권)** 표시
- 등급 기준은 관리자 설정에서 점수 구간별로 조정 가능
- 예시 기준 (확정 전 임시):

| 등급 | 종합점수 구간 | 의미 |
|------|------------|------|
| 하위권 | 0 ~ 59점 | 평균 이하 |
| 중위권 | 60 ~ 79점 | 평균 수준 |
| 상위권 | 80 ~ 89점 | 평균 이상 |
| 최상위권 | 90점 이상 | 멘사 입회 가능성 높음<br>입회테스트 도전 유도 멘트<br>단, 멘사 정책상 개인에게 직접적인 권유는 하지 않으며, 간접적 멘트로 입회테스트 응시 유도 |

> 상향 보정값 적용 후 최종 점수 기준으로 등급 산출

### 5-2. 단계별 점수 계산 공식

```javascript
// 1. 단계별 원점수 계산 (문항ID/보기ID 기반 정답 비교)
단계_원점수 = 기본점수 + (정답개수 × 문제당배점)

// 2. 연령 보정 적용 (1단계 / 2단계 / 3단계)
단계_점수 = 단계_원점수 + 연령보정값
if (단계_점수 > 100) 단계_점수 = 100

// 3. 작업속도 점수 계산
// - 최소소요시간 이하 완료 시 100점
// - 제한시간 내 완료 시 최소 60점 보장 (시간 내 완료 자체를 인정)
// - 최소소요시간 초과 ~ 제한시간 이하: 60~100점 선형 환산
작업속도_원점수 = (실제소요시간 <= 최소소요시간)
  ? 100
  : Math.max(60, 100 - ((실제소요시간 - 최소소요시간) / (제한시간 - 최소소요시간)) * 40)
// 핵심: 범위를 100~60점(40점 폭)으로 설정하여 제한시간 내 완료 시 최소 60점 보장

// 작업속도 연령 보정 적용
작업속도_점수 = 작업속도_원점수 + 연령보정값
if (작업속도_점수 > 100) 작업속도_점수 = 100

// 4. 메타인지 점수 계산 (연령 보정 없음)
// - 나이가 많을수록 경험에 의해 메타인지에 유리하므로 연령 보정 미적용
// - 총 25문제 기준, 문제당 4점 (=100점 만점)
// - 예상 정답수와 실제 정답수의 차이만큼 감점
// - 실제정답수 기준 최대 100점, 오차 1문제당 4점 감점
메타인지_점수 = Math.max(0, 100 - (Math.abs(예상정답수 - 실제정답수) * 4))

// 5. 가중 종합점수 계산 (1단계 가중치 3, 나머지 각 1, 합산 /7)
종합점수_raw = (1단계점수 × 3 + 2단계점수 × 1 + 3단계점수 × 1
              + 작업속도점수 × 1 + 메타인지점수 × 1) / 7

// 6. 전체 점수 상향 보정 (관리자 설정값, 최대 100점 제한)
종합점수_최종 = Math.min(100, 종합점수_raw + 상향보정값)
```

## 최소소요시간 계산법
 
```
단계별 최소소요시간(초) = 최소 평균응답시간(초) × 문항수
전체 최소소요시간(초)   = 1단계 + 2단계 + 3단계

| 2단계 | 24초 | 5문항 | 120초 |
| 3단계 | 24초 | 5문항 | 120초 |
| **합산** | | | **540초** |

## 제한시간 계산법
 
```
단계별 제한시간(초) = 관리자 설정값(분) × 60
전체 제한시간(초)   = 1단계 + 2단계 + 3단계
```
 
| 단계 | 관리자 설정값 | 결과 |
|---|---|---|
| 1단계 | 30분 | 1800초 |
| 2단계 | 10분 | 600초 |
| 3단계 | 10분 | 600초 |
| **합산** | | **3000초** |

## 작업속도 점수 계산법
 
```
점수 범위 폭 = 최고점(100) - 최저점(60) = 40  ← 고정값
 
감점량 = (실제소요시간 - 최소소요시간) ÷ (제한시간 - 최소소요시간) × 점수 범위 폭(40)
원점수 = 100 - 감점량
최종점수 = min(100, 원점수 + 연령보정값)
```
 
> 점수 범위는 60~100점 (40점 폭 고정).  
> 빠를수록 감점이 적고, 느릴수록 감점이 커집니다.

```

= (276 + 96 + 88 + 76 + 96) / 7

≈ 90.3점

## 6. 번호 생성 규칙

### 6-1. 응시번호 생성 규칙 (인증번호 동일 사용)

**개요**

- 시험 응시 시 생성되며, 상세 리포트 결제 시 인증번호로 동일하게 사용

**형식**

```
BC-{년도}-{일련번호 6자리}
예시: BC-2026-001001
```

**생성 규칙**

| 항목 | 설명 |
|------|------|
| 시험 코드 | `BC` (Brain Challenge 고정) |
| 년도 | 응시 연도 4자리 |
| 일련번호 | 년도별 증가 일련번호 + 1000 시작 후 6자리 0 패딩 처리 |
| 초기화 | 매년 1월 1일 기준으로 일련번호 초기화 (1001부터 재시작) |
| 생성 시점 | 시험 응시 시 생성 |

**예시**

```
2026년
BC-2026-001001
BC-2026-001002
  ...

2027년 (연도 변경 시 일련번호 초기화)
BC-2027-001001
BC-2027-001002
  ...
```

### 6-2. 진위확인코드 생성 규칙

**형식**

```
{4자리 해시}-{4자리 해시}-{일련번호 8자리}
예시: A3F2-8D1C-00001001
```

**구성 상세**

| 구성 | 설명 |
|------|------|
| 앞 8자리 (해시) | MD5(이메일 + 응시번호 + 타임스탬프).substring(0, 8).toUpperCase() → `XXXX-XXXX` 형식으로 변환 |
| 뒤 8자리 (일련번호) | 진위확인코드 생성 시점 기준 전역 증가 시퀀스 (1001부터 시작) → 8자리 0 패딩 처리 |

**생성 규칙**

- 일련번호는 연도와 무관하게 계속 증가 (전역 시퀀스)
- 생성 시점: 시험 결과(기본 또는 상세) 조회 시 **최초 1회 생성**
- 이미 생성된 경우 재생성하지 않고 기존 코드 사용

**예시**

```
A3F2-8D1C-00001001
B1D4-92AF-00001002
9F21-AB77-00001003
  ...
```

**95점 이상 - 최상위 수준**

매우 뛰어난 문제 해결 능력과 논리적 사고력을 보유하고 계십니다. 복잡한 패턴을 빠르게 파악하고, 추상적인 개념을 효과적으로 처리하는 능력이 탁월합니다.

**공식 멘사 입회 테스트 합격 가능성이 매우 높습니다.** 온라인 테스트로 확인된 이러한 능력을 멘사코리아 입회테스트를 통해 객관적으로 검증받고, 전 세계 상위 2% 고지능자 모임에 합류해보시기 바랍니다.

**90~94점 - 매우 우수**

탁월한 논리적 사고력과 뛰어난 패턴 인식 능력을 가지고 계십니다. 어려운 문제 상황에서도 체계적으로 접근하여 해결책을 찾아내는 능력이 우수합니다.

**멘사 입회 기준에 충분히 도달할 수 있는 잠재력을 보유하고 있습니다.** 온라인 테스트로 확인한 능력을 멘사코리아 입회테스트를 통해 검증받아보시는 것을 추천드립니다.

**85~89점 - 우수**

평균보다 훨씬 높은 수준의 논리적 사고와 문제 해결 능력을 보유하고 있습니다. 새로운 유형의 문제를 접했을 때 적응력이 빠르며, 체계적인 분석과 추론이 가능합니다.

지속적인 학습과 훈련을 통해 최상위권으로 도약할 수 있는 충분한 잠재력을 갖추고 있습니다. **훈련 후 멘사코리아 입회테스트에 도전해보시기를 권장합니다.**

**75~84점 - 평균 상위**

평균 이상의 안정적인 실력을 보유하고 계십니다. 기본적인 논리 구조를 잘 이해하고 있으며, 일반적인 난이도의 문제는 무난하게 해결할 수 있는 능력을 갖추고 있습니다.

추가적인 학습과 다양한 유형의 문제 풀이 훈련을 통해 더욱 발전할 수 있습니다. 실력 향상 후 **멘사코리아 입회테스트에 도전하여 고지능자 인증을 받아보시는 것도 좋은 목표가 될 수 있습니다.**

**65~74점 - 평균 중상**

안정적인 기본기를 갖추고 있으며, 표준적인 문제 해결 능력을 보유하고 있습니다. 평균 수준의 논리적 사고력을 가지고 있어, 기본 개념을 활용한 문제는 잘 풀어낼 수 있습니다.

꾸준한 훈련과 체계적인 학습으로 상위권 진입이 충분히 가능합니다. 목표를 설정하고 단계적으로 실력을 향상시켜 나가시기 바랍니다.

**55~64점 - 평균**

기본적인 문제 해결 능력을 갖추고 있습니다. 익숙한 유형의 문제는 해결할 수 있으나, 새로운 패턴이나 복잡한 구조의 문제에서는 어려움을 느낄 수 있습니다.

기본 논리 훈련과 패턴 학습을 단계별로 체계적으로 진행하시면 실력 향상에 큰 도움이 됩니다. 꾸준한 학습을 통해 한 단계씩 발전해 나가시기 바랍니다.

**54점 이하 - 성장 단계**

현재는 기초를 다지는 단계로, 앞으로 발전 가능성이 충분합니다. 문제 유형에 대한 이해를 높이고, 기본적인 논리 구조를 익히는 것이 우선입니다.

쉬운 문제부터 차근차근 풀어나가며 자신감을 쌓는 것을 추천합니다. 작은 성공 경험을 반복하며 점진적으로 실력을 키워나가시기 바랍니다.

#### ▶ 개선 가이드 (구체적이고 실행 가능한 조언)

**90점 이상 - 전문성 개발 단계**

- **고급 문제 해결 훈련:** IMO(국제수학올림피아드), 페르미 추정, 논리 퍼즐 챔피언십 수준의 문제에 도전해보세요. 단순 반복보다는 새로운 유형의 난제를 접하는 것이 중요합니다.
- **속도와 정확도의 균형:** 현재 작업 속도 점수를 고려할 때, 빠른 판단력을 유지하면서 실수를 최소화하는 연습 또는 시간 압박 하에서도 정확도를 유지하는 훈련이 필요합니다.
- **메타인지 정교화:** 자기 평가 점수를 개선하기 위해, 문제별 난이도와 소요 시간을 예측하고 기록하는 습관을 들이거나, 자신의 능력을 정확히 파악하여 학습 전략을 스스로 최적화하세요.
- **실전 적용:** 멘사 입회테스트, 논리력 경시대회, 또는 전문 분야(프로그래밍 알고리즘, 데이터 분석 등)에서 실력을 검증받고 활용해보세요.

**80~89점 - 최상위권 도약 전략**

- **약점 영역 집중 공략:** 취약 영역(시각 추론: 공간 지각력 훈련(3D 퍼즐, 회전 문제), 패턴 변환 연습 / 논리 사고: 기호 논리학 기초, 논리 게임(스도쿠, 켄켄), 삼단논법 훈련 / 종합 인지: 복합 문제 풀이, 다단계 추론 연습, 정보 통합 훈련)을 특화 훈련하세요.
- **문제 유형 다양화:** Raven's Progressive Matrices, WAIS 스타일 문제, 추상적 추론 테스트 등 다양한 형식의 고난도 문제를 풀어보세요.
- **시간 관리 최적화:** 작업 속도를 향상시키기 위해, 단계별 시간 제한을 설정하고(1단계 1분 30초, 2단계 1분, 3단계 1분 20초) 반복 연습하거나, 현재 속도를 유지하면서 정확도를 높이는 데 집중하세요.
- **오답 분석 루틴:** 틀린 문제는 단순히 정답 확인이 아니라, '왜 틀렸는지', '어떤 사고 과정이 부족했는지' 분석하고 기록하세요.

**70~79점 - 상위권 진입 로드맵**

- **단계별 학습 플랜 (4주 기준):**
  - 1주차: 기본 패턴 복습 (하루 20분, 쉬운 문제 30개)
  - 2주차: 중급 문제 도전 (하루 30분, 중급 문제 20개)
  - 3주차: 약점 영역 집중 (취약 영역 특화 훈련)
  - 4주차: 모의 테스트 (실전과 동일한 조건으로 3회 이상)
- **영역별 맞춤 훈련:** 각 영역별로(시각 추론: 도형 규칙 찾기, 거울상/회전 문제 집중 / 논리 사고: 조건 추론, 명제 논리, 벤다이어그램 연습 / 종합 인지: 복합 정보 처리, 다중 규칙 적용 문제) 집중 학습하세요.
- **실전 감각 유지:** 매주 1회, 시간 제한을 둔 모의 테스트로 실전 감각을 유지하세요. 점수보다는 '어떤 유형에서 시간이 오래 걸리는지' 파악이 중요합니다.
- **학습 기록 작성:** 일일 학습 내용, 취약 유형, 개선 사항을 간단히 기록하면 학습 효율이 30% 이상 향상됩니다.

**69점 이하 - 체계적 실력 향상 가이드**

- **기초 다지기 (2주 프로그램):**
  - 매일 10-15분, 가장 기본적인 패턴 문제부터 시작
  - 정답률 80% 이상 달성 시 다음 단계로 이동
  - 틀린 문제는 다음날 다시 풀어보기 (반복학습)
- **핵심 유형 마스터:** 취약 영역에서 자주 나오는 5가지 핵심 패턴(회전, 대칭, 중첩, 크기 변화, 개수 규칙 또는 조건 추론, 대소 비교, 순서 배열, 참/거짓 판단, 논리 연산 또는 단계적 추론, 규칙 결합, 예외 찾기, 역추론, 정보 통합)을 집중 학습하세요.
- **컨디션 관리:** 두뇌 활동은 컨디션의 영향을 크게 받습니다. 충분한 수면(7-8시간), 규칙적인 운동, 스트레스 관리 후 재응시하면 10-15점 향상을 기대할 수 있습니다.
- **학습 리소스 활용:** 무료 논리 퍼즐 앱(Lumosity, Peak, Elevate), 유튜브 IQ 테스트 해설 영상, 온라인 논리 게임 등을 활용하세요.
- **작은 성공 경험 쌓기:** 처음에는 쉬운 문제를 빠르게 맞혀 자신감을 쌓고, 점진적으로 난이도를 높이는 전략이 효과적입니다.

1. **시간 감각 훈련:** 스톱워치 없이 30초, 1분을 체감으로 측정하는 연습
2. **선별 전략:** 처음 5초 안에 쉬운 문제와 어려운 문제 구분, 쉬운 것부터 해결
3. **단계적 시간 단축:** 같은 유형 문제를 반복하며 매번 10% 시간 단축 목표 설정

3. **복기 루틴:** 테스트 직후 "몇 개 맞췄을 것 같은지" 예측하고 실제 결과와 비교 분석

## 11. 관리자 기능

### 11-1. 테스트 설정 관리

**단계별 설정 항목 (관리자 페이지 `test-settings.html`)**

| 설정 항목 | 설명 |
|----------|------|
| 출제 문제 수 | 각 단계별 출제할 문제 수 |
| 기본 점수 | 단계별 기본 부여 점수 |
| 문제당 배점 | (100 - 기본점수) / 문제수로 자동 계산 또는 직접 입력 |
| 제한 시간 | 단계별 최대 응시 시간 (분) |
| 최소 소요 시간 | 비정상 패턴 차단용 최소 응답 시간 기준 (초) |
| 최소 정답률 | 비정상 패턴 차단용 최소 정답률 기준 (%) |
| 상향 보정값 | 종합점수 산출 후 추가 적용할 전체 상향 보정값 (점) |

### 11-2. 문제 관리

**기능 목록**

- 단계별 문제 등록 / 수정 / 삭제
- 단계별 문제 은행 현황 조회 (등록 수 확인)

**문제 등록 규칙**

- 문제(질문)는 텍스트만 / 이미지만 / 텍스트+이미지 중 **1가지 형식** 선택
- 보기(선택지)는 **이미지 파일 첨부만** 가능 (텍스트 입력 불가)
- 보기 개수는 최소 2개 ~ 최대 8개, 미사용 보기 입력란은 자동 비활성화
- 정답은 라디오 버튼으로 1개만 선택
- 이미지 첨부 시 등록/조회/수정 모든 화면에서 썸네일 노출
- 채점은 문항 ID와 보기 ID 기반으로 수행 (보기 순서가 랜덤 셔플되어도 ID로 정답 식별)

### 11-3. 응시 로그 관리

**저장 항목**

| 항목 | 설명 |
|------|------|
| 로그 ID | 자동 증가 (PRIMARY KEY) |
| 응시 일시 | 타임스탬프 |
| IP 주소 | 자동 수집 |
| 회원 ID | 로그인 시 저장 |
| 이름 | 사용자 입력 |
| 출생연도 | 사용자 입력 |
| 이메일 | 사용자 입력 |
| 응시번호 | BC-{년도}-{일련번호 6자리} |

## 12. 문의하기 시스템

### 12-1. 변경 사항

**기존 (v1)**

- 로그인 후 문의 가능
- 문의 내역 DB 저장

**1차 변경 (v2)**

- **폼메일 방식**으로 변경
- 로그인 불필요
- 문의 양식 작성 → 관리자 이메일로 자동 전송
- 관리자 페이지에서 수신 이메일 변경 가능


**2차 변경 (v3)**

- **폼메일 방식**
- 로그인 없이 문의 가능
- 문의 양식 작성 후 관리자 이메일로 자동 전송
- 별도의 문의 내역 저장 없음 (DB 미사용)


### 12-2. 문의 양식
┌─────────────────────────────┐
│ 문의 유형:                   │
│  → 온라인테스트 문의 (고정값)   │
│ 이름: [텍스트 입력]           │
│ 전화번호: [선택 + 입력]        │
│ 이메일: [아이디 + 도메인 선택]  │
│                             │
│ 문의 제목: [텍스트 입력]        │
│                             │
│ 문의 내용:                   │
│ [────────────────────────]  │
│ │                        │  │
│ │                        │  │
│ [────────────────────────]  │
│                             │
│ 첨부파일: 최대 3개 (선택)       │
│                             │
│ 개인정보 수집 및 이용 동의 (필수)│
│                             │
│ [목록보기]   [등록하기]         │
└─────────────────────────────┘
**입력 항목 상세**
- 문의 유형: 온라인테스트 문의 (고정값)

- 전화번호 (필수)
- 이메일 (필수)
- 문의 제목 (필수)
- 문의 내용 (필수)
- 첨부파일 (선택, 최대 3개)
- 개인정보 수집 및 이용 동의 (필수)
