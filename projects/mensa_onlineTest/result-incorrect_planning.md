- **프로젝트**: 멘사코리아 온라인테스트
- **담당 PM**: 배은아
- **상태**: 방향 확정 · 기획 진행 중

## 기획 방향 요약

| 항목 | 확정 방향 |
|------|-----------|
| 결제 구조 | 상세리포트+오답확인 번들 상품 (단독 결제 불가) |

## 01. 상품 구성 및 결제 구조

**확정**

테스트 완료 후 결제 페이지에서 3가지 상품을 선택합니다.  
오답 확인은 상세리포트와 번들 구성으로만 제공되며, 단독 결제는 불가합니다.

**사용자 흐름**
> 테스트 완료 → 결제 페이지 → 상품 선택 → 결제 완료 → 결과 화면 열람

**상품 구성**

| # | 상품명 | 판매가 | 원가 | 포함 내용 |
|---|--------|--------|------|-----------|
| ① | 기본결과 | 14,900원 | 19,900원 | 종합 점수, 기본 해석, 이메일 발송 |
| ② | 상세리포트 | 23,900원 | 39,900원 | 기본결과 포함, 영역별 분석, 그래프, PDF |
| ③ | **상세리포트 + 오답확인** | **28,900원** | **49,800원** | 상세리포트 전체 + 단계별 오답 문제·선택 답안 확인 |

**주요 정책**
- 오답 확인 **단독 결제 불가** — 반드시 상세리포트와 번들로만 구매 가능
- 상품 간 복수 선택 불가 (1회 1상품)
- 기본결과 → 상세리포트 업그레이드 시 차액(9,000원) 결제

## 02. 오답 문제 노출 범위

**확정**

틀린 문제의 **문제 이미지 + 사용자가 선택한 보기 이미지**를 함께 노출합니다.

| 항목 | 내용 |
|------|------|
| 노출 항목 | 단계별 틀린 문항 수(00/00) 표시 후 해당 단계 문제 묶음 노출. 틀린 문제 이미지 및 내가 선택한 보기 이미지 제공. 문제 번호는 별도 표시 없이 단계 내 순차 번호로 대체 |
| 미노출 항목 | 정답 보기 이미지, 정답 표시 |
| 정렬 기준 | 단계별 → 문항 번호 순 (오름차순) |
| 단계 타이틀 | 관리자 설정값 표시 (예: 패턴인식, 논리사고, 종합인지) |
| 유출 방지 | 워터마크 삽입, 우클릭 방지, 캡쳐 방지, 드래그 방지 필수 / 안내 문구 노출 |

**워터마크 규격**
- 구성: 응시번호 + 이름
- 방식: 대각선 반복 삽입 (투명도 15~20%)
- 처리: 서버사이드 이미지 처리 권장 (캡처 시에도 워터마크 유지)

## 03. 상품 금액

| 상품 | 판매가 | 원가 | 할인율 |
|------|--------|------|--------|
| 기본결과 | 14,900원 | 19,900원 | 25% |
| 상세리포트 | 23,900원 | 39,900원 | 40% |
| 상세리포트 + 오답확인 | **28,900원** | 49,800원 | **42%** |

> 오답 확인 단독 원가 기준 9,900원에서 상세리포트(23,900원)와 번들 시 5,000원 추가로 제공.  
> 최종 금액은 클라이언트(멘사코리아) 검토 후 확정 요청드립니다.

## 04. 결과 제공 형태

**확정: 웹 화면 열람만**

| 항목 | 제공 여부 |
|------|-----------|
| 웹 열람 | ✅ 제공 |
| 이메일 발송 | ❌ 미제공 |
| PDF 다운로드 | ❌ 미제공 (오답 확인 화면 한정) |
| 링크 발송 | ❌ 미제공 |

> 오답 확인 화면은 웹 열람 전용. 상세리포트 PDF는 기존대로 제공.

## 05. 재열람 가능 여부

**확정: 재열람 불가 (1회 열람)**

| 항목 | 내용 |
|------|------|
| 열람 방식 | 결제 완료 후 즉시 웹 화면에서 1회 열람 |
| 이메일 링크 | 발송 안 함 |
| 재접근 시도 | 열람 완료 처리 후 재접근 차단 |
| 세션 이탈 | 클라이언트 정책 재확인 필요 ⚠️ |

> **기획 이슈**: 세션 이탈(창 닫기, 네트워크 오류)로 실제 미확인 시 예외 처리 정책 클라이언트와 사전 합의 필요  
> 예: 결제 완료 후 5분 이내에 한해 재접근 1회 허용. 이후 차단.

## 06. 보정점수 관련 안내 문구

**확정: 안내 문구 추가**

> "본 테스트의 점수는 단순 정답 수가 아닌, 문제별 난이도 등 복합적인 요소를 반영한 보정 방식으로 산출됩니다. 이에 따라 동일한 정답 수라도 점수가 다를 수 있음을 안내드립니다."

| 항목 | 내용 |
|------|------|
| 노출 위치 | 결과 화면 하단 |
| 노출 대상 | 오답 확인 구매자 전체 |
