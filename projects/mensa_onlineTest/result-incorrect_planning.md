# 멘사코리아 온라인테스트 오답 확인 기능 추가 기획서

- **프로젝트**: 멘사코리아 온라인테스트
- **담당 PM**: 배은아
- **상태**: 방향 확정 · 기획 진행 중

---

## 기획 방향 요약

| 항목 | 확정 방향 |
|------|-----------|
| 결제 구조 | 3번째 독립 상품 (단독 결제 가능) |
| 오답 노출 범위 | 단계별 해당 문제 이미지 + 선택한 보기 이미지 |
| 결과 제공 형태 | 웹 화면 열람 전용 (PDF/이메일 없음) |
| 재열람 | 불가 (1회 열람) |
| 유출 방지 | 워터마크 삽입 (수험번호 + 이름), 드래그 방지, 캡쳐 방지, 우클릭 방지 |
| 보정점수 안내 | 난이도 배점 차이 고지 문구 추가 |

## 01. 오답 확인 상품 노출 시점 및 결제 구조

**확정**

테스트 완료 후 결제 페이지에서 기존 2개 상품과 함께 3번째 상품으로 오답 확인을 독립 노출합니다. 오답 확인만 단독 결제도 가능합니다.
결제 후 각 화면 간 교차 유입도 지원합니다. 오답 확인 화면에서는 기본결과·상세리포트 구매 안내(결제 상품 화면보다 간소화) 및 결제 바로가기 버튼을 제공하며, 기본결과·상세리포트 화면에서도 오답 확인 구매 버튼을 노출

**사용자 흐름**
> 테스트 완료 → 결제 페이지 → 상품 선택 → 결제 완료 → 결과/오답 화면 열람

**상품 구성**
- ① 기본결과
- ② 상세리포트
- ③ **오답 확인 (신규)**

**주요 정책**
- 오답 확인 단독 결제 허용 | 복수선택 불가
- 기본결과/상세리포트 구매 없이도 오답 확인 구매 가능

## 02. 오답 문제 노출 범위

**확정**

틀린 문제의 **문제 이미지 + 사용자가 선택한 보기 이미지**를 함께 노출합니다.

| 항목 | 내용 |
|------|------|
| 노출 항목 | 단계별 틀린 문항 수(00/00) 표시 후 해당 단계 문제 묶음 노출. 틀린 문제 이미지 및 내가 선택한 보기 이미지 제공. 문제 번호는 별도 표시 없이 단계 내 순차 번호로 대체 |
| 미노출 항목 | 정답 보기 이미지, 정답 표시 |
| 정렬 기준 | 단계별 → 문항 번호 순 (오름차순) |
| 유출 방지 | 워터마크 삽입, 우클릭 방지, 캡쳐 방지, 드래그 방지 필수/ 문제유출방지를 위해 해당 기능이 제한된다는 안내문구 노출 |

**워터마크 규격**
- 구성: 응시번호 + 이름
- 방식: 대각선 반복 삽입 (투명도 15~20%)
- 처리: 서버사이드 이미지 처리 권장 (캡처 시에도 워터마크 유지)

## 03. 오답 확인 상품 금액 제안

| 구분 | 제안 금액 | 근거 |
|------|-----------|------|
| 하한가 | 3,000원 | 기본결과 대비 낮은 가격으로 진입장벽 최소화, 부담 없는 추가 구매 유도 |
| **중간가 (추천)** | **5,000원** | 기본결과(14,900원)의 1/3 수준. 웹 열람 전용·재열람 불가 제약 감안 시 체감 가치와 균형 |
| 상한가 | 9,900원 | 기본결과의 약 2/3 수준. 번들 구성 시 활용 가능 |

**추천 구성안**
- 오답 확인 단독: **5,000원**
- 기본결과 + 오답 확인 번들: **8,000원**

> 재열람 불가 + 웹 열람 전용의 제약을 고려하면 5,000원이 적정선으로 판단됩니다.  
> 최종 금액은 클라이언트(멘사코리아) 검토 후 확정 요청드립니다.

## 04. 결과 제공 형태

**확정: 웹 화면 열람만**

| 항목 | 제공 여부 |
|------|-----------|
| 웹 열람 | ✅ 제공 |
| 이메일 발송 | ❌ 미제공 |
| PDF 다운로드 | ❌ 미제공 |
| 링크 발송 | ❌ 미제공 |

> PDF/이메일 미제공은 문제 유출 리스크 차단과 재열람 불가 정책을 동시에 충족하는 구조입니다.
> 화면 디자인은 기존 온라인테스트(응시자 정보 입력, 결과 화면 등)의 CSS를 준용하여 일관성 있게 구현합니다.

## 05. 재열람 가능 여부

**확정: 재열람 불가 (1회 열람)**

| 항목 | 내용 |
|------|------|
| 열람 방식 | 결제 완료 후 즉시 웹 화면에서 1회 열람 |
| 이메일 링크 | 발송 안 함 |
| 재접근 시도 | 열람 완료 처리 후 재접근 차단 |
| 세션 이탈 | 클라이언트 정책 재확인 필요 ⚠️ |

> **기획 이슈**: 세션 이탈(창 닫기, 네트워크 오류)로 실제 미확인 시 예외 처리 정책 클라이언트와 사전 합의 필요 | 예: 결제 완료 후 5분 이내에 한해 재접근 1회 허용. 이후 차단.

## 06. 보정점수 관련 안내 문구

**확정: 안내 문구 추가**

> "본 테스트의 점수는 단순 정답 수가 아닌, 문제별 난이도 등 복합적인 요소를 반영한 보정 방식으로 산출됩니다. 이에 따라 동일한 정답 수라도 점수가 다를 수 있음을 안내드립니다."

| 항목 | 내용 |
|------|------|
| 노출 위치 | 결과 화면 하단 |
| 노출 대상 | 오답 확인 구매자 전체 |

## 07. 추가 기획 검토 필요 사항

- [ ] **세션 이탈 예외 처리 정책** 클라이언트 확인
- [ ] **가격 최종 확정** (3,000 / 5,000 / 8,000원 중 선택 + 번들 구성 여부)
- [ ] **오답 화면 UI 설계** — 단계별 그룹핑 vs 문항 번호 순, 이미지 크기, 워터마크 렌더링 방식
- [ ] **서버사이드 워터마크 처리 방식** 개발팀 협의
- [ ] **결제 페이지 UI 변경** 범위 확인 (2개 → 3개 상품, 공수 산정)
