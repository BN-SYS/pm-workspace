Description — 온라인테스트 문의내역 상세
  • 우측 상단 배지에 현재 문의의 답변상태가 표시되며, 관리 영역의 답변상태 드롭다운 변경 저장 시 연동된다.
  • 문의 내용(기본정보 · 문의내용)은 읽기 전용으로 노출된다.
  • 첨부파일은 파일명 클릭 시 즉시 다운로드된다.
  • 관리 영역에서 답변상태를 드롭다운으로 선택할 수 있다. 옵션: 답변중(기본) / 중간답변 / 답변완료
  • 답변내용 입력 후 [저장] 클릭 시 확인 창("해당 내용으로 메일이 발송됩니다. 발송하시겠습니까?")이 표시되며, 확인 클릭 시 저장 및 문의자 이메일로 답변이 자동 발송된다. 발신메일: 멘사코리아 시스템 발송용 메일로 발송 / 메일 제목: [MENSAKOREA] 온라인테스트 문의 답변입니다. / 메일 내용: 입력한 답변 내용
  • 답변 등록 완료 후, 답변내용 옆에 등록자명 · 등록일시(yyyy-mm-dd hh:mm:ss)가 표기되며, 수정 시 최종 수정 정보로 업데이트된다.
  • 답변 등록 후 [저장] 버튼은 [수정] 버튼으로 전환되며, 수정 버튼 클릭 시 기존 답변 내용이 로드되어 수정 입력이 가능해진다. 수정 저장 클릭 시에도 동일한 확인 창이 표시되며, 확인 클릭 시 변경된 내용으로 메일이 재발송된다. [취소] 클릭 시 수정을 중단하고 기존 등록된 답변 내용 조회 상태로 되돌아간다.
  • 담당자 메모는 내부 관리 전용으로, 사용자에게는 어떠한 방식으로도 노출되지 않는다. 메모는 저장 시 이전 내용을 덮어쓰는 업데이트 방식으로 운영된다.
  • [삭제] 버튼 클릭 시 confirm 모달 표시 후 확인 시 해당 문의 전체가 삭제되며, 삭제된 데이터는 복구되지 않는다.
