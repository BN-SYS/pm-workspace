Description — 온라인테스트 문의내역 목록
  • 사용자 화면에서 제출된 온라인테스트 문의(비회원도 가능)를 관리하기 위해 기존 문의 관리 메뉴에 하위 메뉴 '온라인테스트 문의내역' 을 추가.
  • 검색 필터 레이아웃은 기존 문의 내역 화면과 동일하며, 조건검색의 검색기준은 드롭다운은 이름 / 이메일 / 전화번호 / 제목으로 적용.
  • 문의하기와 동일하게 답변상태에 따라 행 컬러를 다르게 적용(답변중-#FFE0DC,중간답변-#FFFED4,답변완료-흰색바탕)
  • 체크박스 선택 후 [선택삭제] 버튼을 통해 일괄 삭제 가능.
  • 행 클릭시 해당 문의 상세 페이지 이동.

    사용자 문의폼 디스크립션

  • 사용자 화면(https://test.mensakorea.kr/onlinetest-Inquiry-form.php)
  • 응시정보인 이름·출생연도·이메일은 필수 입력 항목이다. 출생연도 드롭다운: 테스트응시정보 입력과 동일하게 현재 연도 기준 -8년 ~ 1930년까지 자동 생성 (JS로 초기화)
  • 유효성 검사는 제출 버튼 클릭 시 실행된다. 오류 발생 시 해당 필드에 빨간 테두리와 오류 메시지가 인라인으로 표시되며, 첫 번째 오류 필드로 자동 스크롤·포커스된다. 필수 항목: 이름 / 출생연도 / 이메일 / 문의 제목 / 문의 내용 / 개인정보 동의
